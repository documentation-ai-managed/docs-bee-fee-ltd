## Getting Started

Launch your Bee-Fee integration in minutes. Follow these steps to go live.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your account at https://platform.beefee.com/register. Verify your email to receive your API key.
  </Step>
  <Step title="Integrate API" icon="code">
    Use our SDKs for quick setup. Authenticate with your `{API_KEY}`.

    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      const BeeFee = require('beefee-sdk');
      const client = new BeeFee({ apiKey: 'YOUR_API_KEY' });
      const games = await client.games.list();
      console.log(games);
      ```
      ```python
      import beefee
      client = beefee.Client(api_key='YOUR_API_KEY')
      games = client.games.list()
      print(games)
      ```
    </CodeGroup>
  </Step>
  <Step title="Launch" icon="rocket">
    Test in sandbox mode, then switch to production. Monitor via our dashboard.
  </Step>
</Steps>
